# Accessibility Testing Skill

## WCAG 2.1 Levels
- A (minimum)
- AA (recommended)
- AAA (enhanced)

## Tools
- axe-core (automated)
- Lighthouse accessibility audit
- Screen reader testing (NVDA, VoiceOver)
- Keyboard navigation testing

## Common Issues
- Missing alt text
- Insufficient color contrast
- Missing form labels
- Focus management
